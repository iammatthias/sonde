# Sonde

ATProto Record Explorer. Browse identities, collections, and records across the AT Protocol network.

**https://sonde.blue**

## What is this?

A tool for exploring [AT Protocol](https://atproto.com) data. Look up handles and DIDs, browse collections, view decoded records with clickable references.

Named after *radiosonde* — instrument packages carried by weather balloons to measure the atmosphere.

## Built with

- [Astro](https://astro.build) — Static & SSR framework
- [Svelte](https://svelte.dev) — UI components
- [@atproto/api](https://atproto.com) — AT Protocol client
- [Constellation](https://docs.bsky.app/docs/advanced-guides/constellation) — Network statistics
- [Cloudflare Pages](https://pages.cloudflare.com) — Deployment
